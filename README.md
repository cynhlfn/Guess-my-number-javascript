# 🎲 Guess My Number Game

## Description

Ceci est un mini-jeu de devinettes en JavaScript, réalisé dans le cadre de la formation Udemy.

Le but du jeu est simple : deviner un nombre secret entre 1 et 100.

### La page présente :

- Un champ pour entrer le nombre deviné
- Un bouton Check! pour vérifier le nombre
- Un bouton Again! pour recommencer la partie
- Un affichage du score actuel et du highscore
- Des messages interactifs pour guider le joueur : trop haut, trop bas, correct ou perdu

Le jeu utilise JavaScript pur pour gérer la logique, et HTML/CSS pour la structure et le style.

## Technologies utilisées

- HTML5
- CSS3
- JavaScript (ES6)
- Google Fonts (Press Start 2P)

## Compétences développées

- Manipulation du DOM avec document.querySelector
- Gestion des événements (addEventListener)
- Logique conditionnelle en JavaScript
- Mise à jour dynamique du contenu HTML
- Styling CSS moderne et typographie rétro
- Gestion du score et du highscore
- Fonctionnalités du jeu
- Génération aléatoire d’un nombre secret entre 1 et 100
- Messages interactifs selon la valeur entrée par le joueur
- Réduction du score à chaque mauvaise réponse
- Mise à jour du highscore si le joueur fait mieux
- Bouton “Again!” pour recommencer sans recharger la page
- Style responsive et visuel rétro inspiré des jeux vidéo
